Отчет по этапу проекта с 23.03.2020 по 05.04.2020

Запланировано: Разработка Buisness Requirements Document. Разработка детализированного плана проекта с основными эпохами. Назначение Ролей. Создание проектных диаграмм.

Выполнено: 
	1. Создан документ с требованиями к проекту, описана основная мотивационная идея релизации проекта.
		Артефакт: документ с требованиями к проекту.
	2. Создан план проекта с опсианием основных этапов ЖЦ проекта. Создан подробный план с основными этапами и задачами разработки,с назначенными датами и ролями.
		Артефакт: План-документ проекта
	3. Создана блок-схема логики работы проекта, включающая осноные шаги, препринимаемые пользователем при работе с приложением. 
		Артефакт: Блок-схема, включенная в план-документ проекта
	4. Создана core диаграмма классов проекта.
		Артефакт: диаграмма классов

Возникшие проблемы: 
	Возникновение тупиковых моментов в описании логики работы проекта по причине недостаточного анализа имеющихся ресурсов, в частности API для сайта аптеки.
		Решение: возвращение пользователя из тупиковых моментов в основную точку входа приложения. 
	Нарушение запланированных сроков выполнения проекта в связи с нехваткой времени у участников проекта по причине наличия дургих проектов, не касаемых данного:
		Решение: Сокращение второстепенных запланированных задач, с целью реализации основного функционала вовремя.
